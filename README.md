# ConsoleMinesweeper
Minesweeper in the C# console. What could be better? Maybe [2048](https://github.com/IanWold/Console2048), [a maze](https://github.com/IanWold/ConsoleMazeWoot), or maybe [a game engine](https://github.com/IanWold/OutrageEngine).

## How To
Type the row/column index of a square to "press" it. Type an "f" in front to flag it. Example:

```
>: g5

>: f g5
```

Pretty easy.
